# 【概述】

## 1、基于人脸识别的驾驶员状态预警系统

  人脸识别+面部表情识别

  根据驾驶员的面部表情判断驾驶员的驾驶状态

  使用OpenCV对图像中的人脸进行识别

  对识别后的人脸使用keras进行面部表情识别（CNN模型）
  
  
  
## 2、微笑打卡系统（更新中）

  人脸识别+活体检测+面部识别
  
  在门禁系统的基础上增加面部表情识别
